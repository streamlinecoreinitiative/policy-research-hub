Model: Qwen 3.5 — 4B params

Senior Writer & Editor

Drafts publication-grade policy analysis, then performs an editorial pass — applying fact-check flags, tightening prose, and auditing sources. Two-pass workflow for quality.

Model: Qwen 3.5 — 9B params

Verifies every statistical claim against source documents. Flags unsupported assertions, prevents data fabrication. Also runs a deterministic QA gate on word count, sections, and readability.

Architecture Highlights

🖥️ Hardware

Apple M4 · 16 GB unified memory · 100% GPU inference · Local-first — no cloud API costs for inference

🔄 Scheduler Engine

Persistent in-memory scheduler with 60s tick loop. Survives hot reloads. Auto-retry with exponential backoff. Concurrent run protection via mutex.

🛡️ Reliability

Streaming Ollama calls (no idle timeouts). Automatic model fallback. Write mutex on article index. Filesystem-safe for serverless.

📡 Distribution

Auto git commit → GitHub push → Vercel deploy. Google Drive HTML upload. 3-platform social media queue (Twitter, LinkedIn, Bluesky).

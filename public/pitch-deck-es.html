Aspectos Destacados de Arquitectura

🖥️ Hardware

Apple M4 · 16 GB memoria unificada · 100% inferencia en GPU · Local-first — sin costos de API en la nube para inferencia

🔄 Motor de Scheduler

Scheduler persistente en memoria con ciclo de 60s. Sobrevive hot reloads. Reintento automático con backoff exponencial. Protección contra ejecuciones concurrentes vía mutex.

🛡️ Confiabilidad

Llamadas streaming a Ollama (sin timeouts de inactividad). Fallback automático de modelos. Mutex de escritura en índice de artículos. Compatible con filesystem serverless.

📡 Distribución

Auto git commit → push a GitHub → deploy en Vercel. Subida HTML a Google Drive. Cola de redes sociales en 3 plataformas (Twitter, LinkedIn, Bluesky).
